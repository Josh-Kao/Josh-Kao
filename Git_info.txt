*** 在 Github(web) 建立 repository Josh-Kao
============================================
會更新 SAP*.ini 的電腦上

1) 在 Gitbash 建立 & 進入目錄 Josh-Kao
2) git clone https://github.com/Josh-Kao/Josh-Kao.git (URL copy from GitHub)
3) 將 sap*.init copy 進此目錄
4) git add *.ini
4) git commit -m "20220311"
5) git push 

在要下載的電腦
1) 建立目錄 & 進入目錄 Josh-Kao
2) git clone https://github.com/Josh-Kao/Josh-Kao.git
